# Memory Game

A responsive browser-based Memory Game built with **HTML**, **CSS**, and **JavaScript (DOM manipulation)**.

## Features

- 4x4 card board (8 matching pairs)
- Card flip interaction on click
- Randomized card positions using shuffle logic
- Match checking with short delay for non-matching pairs
- Move counter and timer
- Restart button to reset and reshuffle the board
- Responsive layout for desktop and mobile

## Project Structure

- `index.html` – page structure and game containers
- `styles.css` – responsive styling and card flip visuals
- `script.js` – game state, shuffle, flip, match, and restart logic


## How to Play

1. Click a card to flip it and reveal its symbol.
2. Click a second card to try to find a match.
3. If both cards match, they stay face-up.
4. If they do not match, they flip back after a short delay.
5. Keep matching pairs until all cards are revealed.
6. Use **Restart** anytime to start a new shuffled game.
